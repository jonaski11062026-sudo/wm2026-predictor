# WM 2026 Predictor

Eine lokale Single-Page-Web-App zur Simulation der FIFA WM 2026.

## Funktionen

- Gruppen A bis L mit aktueller WM-2026-Gruppeneinteilung
- Ranglisten-Auswahl pro Gruppe ohne Resultateingabe
- automatische Ermittlung der Gruppensieger, Zweitplatzierten und Drittplatzierten
- Auswahl der acht besten Drittplatzierten
- interaktiver K.o.-Baum bis zum Weltmeister
- Spiel um Platz 3
- automatische Speicherung im Browser
- JSON Import und Export
- Hell- und Dunkelmodus

## Nutzung

Die App kann direkt über `index.html` geöffnet werden. Für GitHub Pages reicht es, dieses Repository zu veröffentlichen und Pages für den `main`-Branch zu aktivieren.
